# .dotfiles

Welcome to my .files

## Installation

```bash
git clone https://github.com/Jay-Karia/.dotfiles.git $HOME/.dotfiles
```

## Setup

Change directory

```bash
cd $HOME/.dotfiles
```

```bash
ln -s $HOME/.dotfiles/zshrc/.zshrc $HOME/.zshrc
ln -s $HOME/.dotfiles/git/.gitconfig $HOME/.gitconfig
```
